### Installation instrcutions on Ubuntu ###

1. Preferably use conda env with python 2.7

2. sudo apt-get install -y gfortran build-essential libfftw3-dev libblas3 liblapack3 libarmadillo-dev

3. Make sure these packages are installed in Anaconda env:
numpy
psutil
cython
scipy
scikit-learn

4. python setup.py install
